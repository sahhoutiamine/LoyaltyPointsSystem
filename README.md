# ShopEasy Loyalty System

Système de fidélité MVC avec PHP et Twig

## Installation

1. Cloner le dépôt
2. `composer install`
3. Importer `database/schema.sql` dans MySQL
4. Configurer `config/config.php`
5. Accéder à `public/` via un serveur web

## Structure MVC

- `app/Controllers/` : Contrôleurs
- `app/Models/` : Modèles et logique métier
- `app/Views/` : Templates Twig
- `public/` : Point d'entrée et assets
